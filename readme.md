# 📊 Sales Analytics Dashboard

## Project Overview

This project is an interactive Sales Analytics Dashboard developed using Python, Pandas, Streamlit, and Plotly.

The dashboard helps analyze sales performance through dynamic filters, KPI metrics, and visualizations. Users can explore sales trends by city, category, and date range.

---

## Features

* ✅ City-wise filtering
* ✅ Category-wise filtering
* ✅ Date range filtering
* ✅ Total Sales KPI
* ✅ Top Sales City
* ✅ Highest Profit City
* ✅ Least Sold Product
* ✅ Best Sales Month
* ✅ Interactive Bar Chart
* ✅ Interactive Line Chart
* ✅ Interactive Pie Chart
* ✅ Category data cleaning and correction

---

## Technologies Used

* Python

* Plotly Express

## Dataset

This project uses an Amazon Sales Dataset obtained from Kaggle.

The dataset contains historical sales records from:

**January 2020 to December 2024**

Columns include:

* Order Date
* Product Name
* Category
* Brand
* Quantity
* Total Amount
* Tax
* Shipping Cost
* Discount
* Payment Method
* City
* State
* Country

The dataset was cleaned and processed using Pandas before analysis.

---

## Data Cleaning Performed

* Checked missing values
* Checked duplicate records
* Converted OrderDate to datetime format
* Created Month column for trend analysis
* Calculated Estimated Profit
* Corrected product-category mapping inconsistencies
